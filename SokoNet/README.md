# SokoNet

SokoNet is a Kenyan fintech, marketplace, logistics, jobs, and social commerce ecosystem built as a full-stack enterprise platform.

## Workspace Structure

- `client/` - React + Vite web app
- `server/` - Node.js + Express + MongoDB API
- `mobile/` - Expo React Native mobile app
- `docs/` - Architecture and deployment documentation

## Local Development

1. Install dependencies:
   ```bash
   npm run install:all
   ```

2. Start backend:
   ```bash
   npm run dev:server
   ```

3. Start web client:
   ```bash
   npm run dev:client
   ```

4. Start mobile app:
   ```bash
   npm --prefix mobile start
   ```
